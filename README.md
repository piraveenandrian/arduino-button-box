# Arduino Button Box

## Overview
This is a custom Arduino-based button box for controlling applications (e.g., games, simulators, or media). Each button sends a unique signal to the computer or connected device.

## Components
- Arduino Uno / Nano
- Push buttons
- Breadboard / wires
- Resistors (as needed)

## Wiring Diagram
![Wiring Diagram](wiring-diagram.png)

## Code
Upload `ButtonBox.ino` to your Arduino IDE. Adjust pin assignments if needed.

## Features
- Customizable button mapping
- Easy to extend for more buttons
- Works with Arduino IDE and standard libraries
